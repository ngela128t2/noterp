# NOTERP

회계법인 성지 업무관리 시스템

## 모듈

- 대시보드 + 캘린더
- 거래처 관리 (사업자등록증 AI 인식)
- 프로젝트 + 타임라인
- 데일리 메모 (AI 프로젝트 자동 태깅)
- 매출매입장 (정기거래 자동생성)
- 데이터 가져오기

## 로컬 실행

```bash
npm install
npm run dev
```

`http://localhost:5173` 접속

## 배포

Vercel에 GitHub 연결하면 자동 배포됨.

## AI 기능 사용 (선택)

사업자등록증 자동 인식, 데일리 메모 자동 태깅 기능을 쓰려면
Anthropic API 키가 필요해요.

향후 업데이트에서 환경변수 설정 추가 예정.
